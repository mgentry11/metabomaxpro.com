Definitions

Understanding precise definitions is crucial for constructing valid proofs. Each definition below includes the formal statement, intuitive explanation, and examples.

Set Equality
\(A = B \iff A \subseteq B \text{ and } B \subseteq A\)

What this means: Two sets are equal if and only if they contain exactly the same elements. To prove two sets are equal, you must show that every element in \(A\) is also in \(B\) (this proves \(A \subseteq B\)), AND every element in \(B\) is also in \(A\) (this proves \(B \subseteq A\)).

Why both directions? Just because \(A \subseteq B\) doesn't mean \(A = B\). For example, \(\{1\} \subseteq \{1, 2\}\), but \(\{1\} \neq \{1, 2\}\). You need both containments to ensure the sets have no "extra" elements.

Example: Prove \(\{x : x^2 = 4, x \in \mathbb{Z}\} = \{-2, 2\}\)

(\(\subseteq\)): If \(x^2 = 4\) and \(x \in \mathbb{Z}\), then \(x = 2\) or \(x = -2\). So every element of the left set is in \(\{-2, 2\}\).

(\(\supseteq\)): Check: \(2^2 = 4\) ✓ and \((-2)^2 = 4\) ✓. So \(\{-2, 2\}\) is contained in the left set.

Partition
A partition of a set \(S\) is a collection of subsets \(\{A_i\}_{i \in I}\) such that:
  1. Each \(A_i \neq \emptyset\) (non-empty)
  2. \(A_i \cap A_j = \emptyset\) for \(i \neq j\) (pairwise disjoint)
  3. \(\bigcup_{i \in I} A_i = S\) (union covers all of \(S\))

What this means: A partition "cuts up" a set into non-overlapping pieces that together make up the whole set. Think of it like cutting a pizza into slices—each slice is non-empty, no two slices overlap, and together they form the whole pizza.

Example: Let \(S = \{1, 2, 3, 4, 5, 6\}\). Then \(\{\{1, 2\}, \{3, 4\}, \{5, 6\}\}\) is a partition.

Also: \(\{\text{evens}, \text{odds}\} = \{\{2, 4, 6\}, \{1, 3, 5\}\}\) is a partition of \(S\).

Non-Example: \(\{\{1, 2\}, \{2, 3\}, \{4, 5, 6\}\}\) is NOT a partition because \(\{1,2\}\) and \(\{2,3\}\) both contain 2 (not disjoint).
Equivalence Relation
A relation \(\sim\) on a set \(S\) is an equivalence relation if it satisfies:
  1. Reflexive: \(a \sim a\) for all \(a \in S\)
  2. Symmetric: If \(a \sim b\), then \(b \sim a\)
  3. Transitive: If \(a \sim b\) and \(b \sim c\), then \(a \sim c\)

What this means: An equivalence relation is a way of saying certain elements are "essentially the same" for some purpose. It generalizes the notion of equality.

Key insight: Every equivalence relation on \(S\) creates a partition of \(S\) into equivalence classes, and vice versa!

Example: "Has the same remainder when divided by 3" on \(\mathbb{Z}\).
  • Reflexive: \(n\) has the same remainder as \(n\) ✓
  • Symmetric: If \(n\) and \(m\) have the same remainder, so do \(m\) and \(n\) ✓
  • Transitive: If \(n \sim m\) and \(m \sim k\), all three have the same remainder ✓

This creates 3 equivalence classes: \([0] = \{\ldots, -3, 0, 3, 6, \ldots\}\), \([1] = \{\ldots, -2, 1, 4, 7, \ldots\}\), \([2] = \{\ldots, -1, 2, 5, 8, \ldots\}\)

Partial Order
A relation \(\leq\) on a set \(S\) is a partial order if it satisfies:
  1. Reflexive: \(a \leq a\) for all \(a \in S\)
  2. Antisymmetric: If \(a \leq b\) and \(b \leq a\), then \(a = b\)
  3. Transitive: If \(a \leq b\) and \(b \leq c\), then \(a \leq c\)

What this means: A partial order gives a way to compare elements, but not every pair needs to be comparable. It's called "partial" because some elements may be incomparable.

Key difference from equivalence: Antisymmetric vs Symmetric! Equivalence: \(a \sim b \Rightarrow b \sim a\). Partial order: \(a \leq b\) and \(b \leq a \Rightarrow a = b\).

Example: Subset relation \(\subseteq\) on \(\mathcal{P}(S)\).
  • Reflexive: \(A \subseteq A\) ✓
  • Antisymmetric: \(A \subseteq B\) and \(B \subseteq A \Rightarrow A = B\) ✓
  • Transitive: \(A \subseteq B\) and \(B \subseteq C \Rightarrow A \subseteq C\) ✓

Note: \(\{1\}\) and \(\{2\}\) are incomparable—neither is a subset of the other.

Quotient Set
Given an equivalence relation \(\sim\) on \(S\), the quotient set is: \[S/\!\sim \;= \{[a] : a \in S\}\] where \([a] = \{x \in S : x \sim a\}\) is the equivalence class of \(a\).

What this means: The quotient set is the set of all equivalence classes. It "collapses" equivalent elements into single representatives.

Key property: The quotient set forms a partition of \(S\).

Example: \(\mathbb{Z}/\!\equiv_3\) (integers mod 3).

The quotient set is \(\{[0], [1], [2]\}\), which we often write as \(\mathbb{Z}_3 = \{0, 1, 2\}\).

This has only 3 elements, even though \(\mathbb{Z}\) is infinite!

Continuity
A function \(f\) is continuous at \(c\) if: \[\lim_{x \to c} f(x) = f(c)\]

Equivalently (epsilon-delta): For every \(\varepsilon > 0\), there exists \(\delta > 0\) such that:

\[|x - c| < \delta \Rightarrow |f(x) - f(c)| < \varepsilon\]

What this means: A function is continuous at \(c\) if you can make \(f(x)\) as close to \(f(c)\) as you want by choosing \(x\) sufficiently close to \(c\). Intuitively: no "jumps" or "holes" at \(c\).

For proofs: The epsilon-delta definition is required. Given any \(\varepsilon > 0\), you must find a \(\delta\) (which may depend on \(\varepsilon\) and \(c\)) that works.

Example: Prove \(f(x) = 2x + 1\) is continuous at \(c = 3\).

We need \(|f(x) - f(3)| < \varepsilon\), i.e., \(|2x + 1 - 7| = |2x - 6| = 2|x - 3| < \varepsilon\).

Choose \(\delta = \varepsilon/2\). Then \(|x - 3| < \delta \Rightarrow 2|x-3| < 2\delta = \varepsilon\). ✓

Isomorphism
An isomorphism is a bijective function \(f: A \to B\) that preserves structure: \[f(a \star b) = f(a) \star' f(b)\] where \(\star\) is the operation on \(A\) and \(\star'\) is the operation on \(B\).

What this means: Two structures are isomorphic if they are "essentially the same"—they have the same algebraic properties, just with different labels on the elements.

Requirements: (1) Bijection (one-to-one and onto), (2) Preserves the operation(s).

Example: \((\mathbb{R}, +)\) is isomorphic to \((\mathbb{R}^+, \cdot)\) via \(f(x) = e^x\).
  • Bijection: \(e^x\) maps \(\mathbb{R}\) onto \(\mathbb{R}^+\) one-to-one ✓
  • Preserves structure: \(f(a + b) = e^{a+b} = e^a \cdot e^b = f(a) \cdot f(b)\) ✓
Injective / Surjective / Bijective
Let \(f: A \to B\) be a function.
  • Injective (one-to-one): \(f(a_1) = f(a_2) \Rightarrow a_1 = a_2\)
  • Surjective (onto): For every \(b \in B\), there exists \(a \in A\) with \(f(a) = b\)
  • Bijective: Both injective and surjective

Injective: No two inputs give the same output. "Horizontal line test"—each \(y\)-value hit at most once.

Surjective: Every element of the codomain is hit. The range equals the codomain.

Bijective: Perfect one-to-one correspondence. Has an inverse function.

Examples:
  • \(f: \mathbb{R} \to \mathbb{R}\), \(f(x) = x^2\): Neither (not injective: \(f(-2) = f(2)\); not surjective: -1 not hit)
  • \(f: \mathbb{R} \to \mathbb{R}^+\), \(f(x) = e^x\): Bijective
  • \(f: \mathbb{Z} \to \mathbb{Z}\), \(f(n) = 2n\): Injective but not surjective (odd numbers not hit)

Axioms & Principles

Axioms are fundamental truths we accept without proof. These form the foundation of mathematical reasoning.

Principle of Mathematical Induction
Let \(P(n)\) be a statement about natural numbers. If:
  1. Base Case: \(P(1)\) is true
  2. Inductive Step: For all \(k \geq 1\), \(P(k) \Rightarrow P(k+1)\)
Then \(P(n)\) is true for all \(n \in \mathbb{N}\).

Intuition: Think of dominoes. If the first domino falls (base case), and each domino knocks over the next (inductive step), then all dominoes fall.

Why it works: This is an axiom about natural numbers. It captures the idea that \(\mathbb{N}\) has no "gaps"—you can reach any natural number by starting at 1 and adding 1 repeatedly.

Example: Prove \(\sum_{i=1}^{n} i = \frac{n(n+1)}{2}\).

Base: \(n=1\): \(1 = \frac{1 \cdot 2}{2} = 1\) ✓

Inductive step: Assume true for \(k\). Then:

\(\sum_{i=1}^{k+1} i = \sum_{i=1}^{k} i + (k+1) = \frac{k(k+1)}{2} + (k+1) = \frac{k(k+1) + 2(k+1)}{2} = \frac{(k+1)(k+2)}{2}\) ✓

Zorn's Lemma
If \(S\) is a non-empty partially ordered set in which every chain has an upper bound, then \(S\) has a maximal element.

Key terms:

  • Chain: A totally ordered subset (every two elements are comparable)
  • Upper bound: An element \(\geq\) every element in the chain
  • Maximal element: An element with nothing strictly greater

Equivalence: Zorn's Lemma is equivalent to the Axiom of Choice and the Well-Ordering Theorem.

Usage: Proving existence of maximal ideals, bases for vector spaces, etc.

Axiom of Choice
Given any collection of non-empty sets \(\{A_i\}_{i \in I}\), there exists a function \(f\) (a "choice function") such that \(f(i) \in A_i\) for all \(i \in I\).

What this means: You can always "choose" one element from each set in a collection, even if the collection is infinite.

Why it's controversial: For infinite collections, there's no constructive way to specify which element to pick. The axiom just asserts that a choice function exists.

Consequences: Every vector space has a basis, every set can be well-ordered, Banach-Tarski paradox.

Law of Excluded Middle
For any proposition \(P\): \(P \lor \neg P\) is true.

What this means: Every statement is either true or false—there's no middle ground.

Why it matters: This justifies proof by contradiction. If assuming \(\neg P\) leads to a contradiction, then \(P\) must be true (since one of \(P\) or \(\neg P\) must hold).

Note: Constructive/intuitionistic mathematics rejects this axiom and requires explicit constructions.

Theorems (Statements & Proofs)

These are the key results you should know, along with their proofs.

Cantor's Theorem
Statement: For any set \(A\), \(|A| < |\mathcal{P}(A)|\).

The power set is always strictly larger than the original set.

Intuition: Even for infinite sets, there are "more" subsets than elements. This shows there are different "sizes" of infinity! \(|\mathbb{N}| < |\mathcal{P}(\mathbb{N})| = |\mathbb{R}|\).
Proof (Diagonalization)
First, \(|A| \leq |\mathcal{P}(A)|\) via the injection \(a \mapsto \{a\}\).
We show there's no surjection \(f: A \to \mathcal{P}(A)\). Suppose for contradiction such an \(f\) exists.
Define the "diagonal" set: \(D = \{x \in A : x \notin f(x)\}\).
Since \(D \subseteq A\), we have \(D \in \mathcal{P}(A)\). Since \(f\) is surjective, there exists \(d \in A\) with \(f(d) = D\).
Is \(d \in D\)?
  • If \(d \in D\): By definition of \(D\), \(d \notin f(d) = D\). Contradiction!
  • If \(d \notin D\): By definition of \(D\), \(d \in f(d) = D\). Contradiction!
Both cases lead to contradiction, so no such \(f\) exists. Therefore \(|A| < |\mathcal{P}(A)|\). \(\square\)
Schröder-Bernstein Theorem
Statement: If \(|A| \leq |B|\) and \(|B| \leq |A|\), then \(|A| = |B|\).

Equivalently: If there exist injections \(f: A \to B\) and \(g: B \to A\), then there exists a bijection \(h: A \to B\).

Intuition: If \(A\) "fits inside" \(B\) and \(B\) "fits inside" \(A\), they must be the same size. This is surprisingly non-trivial for infinite sets!

(Proof not required per list, but the theorem is crucial for comparing cardinalities.)

Fermat's Little Theorem
Statement: If \(p\) is prime and \(a\) is any integer, then \(a^p \equiv a \pmod{p}\).

Equivalently, if \(\gcd(a, p) = 1\), then \(a^{p-1} \equiv 1 \pmod{p}\).

Intuition: In modular arithmetic mod \(p\), raising to the \(p\)-th power "wraps around" to give you back the original number. The non-zero elements form a group of order \(p-1\).
Proof (by Induction on \(a \geq 0\))
When \(a = 0\): \(0^p = 0 \equiv 0 \pmod{p}\). ✓
Assume \(k^p \equiv k \pmod{p}\) for some \(k \geq 0\).
We prove \((k+1)^p \equiv k+1 \pmod{p}\).

By the Binomial Theorem:

\((k+1)^p = \sum_{i=0}^{p} \binom{p}{i} k^{p-i} = k^p + \binom{p}{1}k^{p-1} + \cdots + \binom{p}{p-1}k + 1\)

For \(1 \leq i \leq p-1\), \(\binom{p}{i} = \frac{p!}{i!(p-i)!}\). Since \(p\) is prime and \(p\) divides the numerator but not the denominator, \(p \mid \binom{p}{i}\).

Thus all middle terms are \(\equiv 0 \pmod{p}\), so:

\((k+1)^p \equiv k^p + 1 \equiv k + 1 \pmod{p}\) (by IH). ✓ \(\square\)

Russell's Paradox
Statement: There is no "set of all sets."
Intuition: Unrestricted set formation leads to contradictions. This paradox motivated axiomatic set theory (ZFC), which restricts how sets can be formed.
Proof
Suppose for contradiction that \(U\), the set of all sets, exists.
Define \(R = \{A \in U : A \notin A\}\) (sets that don't contain themselves).
Since \(R\) is a set and \(U\) contains all sets, \(R \in U\). Ask: Is \(R \in R\)?
  • If \(R \in R\): By definition of \(R\), \(R \notin R\). Contradiction!
  • If \(R \notin R\): By definition of \(R\), \(R \in R\). Contradiction!
Both cases contradict, so \(U\) cannot exist. \(\square\)
Well-Ordering Principle
Statement: Every non-empty subset of \(\mathbb{N}\) has a least element.
Intuition: You can't go "infinitely downward" in the natural numbers. This is equivalent to the Principle of Induction!
Proof (Using Strong Induction - Contrapositive)

We prove: If \(S \subseteq \mathbb{N}\) has no least element, then \(S = \emptyset\).

Let \(P(n)\) be the statement "\(1, 2, \ldots, n \notin S\)".
Is \(1 \in S\)? If so, 1 would be the least element of \(S\) (since it's the smallest natural number). Since \(S\) has no least element, \(1 \notin S\). So \(P(1)\) is true.
Assume \(P(k)\) is true, meaning \(1, 2, \ldots, k \notin S\).

Is \(k+1 \in S\)? If so, since no smaller number is in \(S\), \(k+1\) would be the least element. But \(S\) has no least element, so \(k+1 \notin S\).

Thus \(P(k+1)\) is true.

By strong induction, \(P(n)\) holds for all \(n\). So \(\forall n \in \mathbb{N}, n \notin S\), meaning \(S = \emptyset\). \(\square\)
Infinitude of Primes
Statement: There are infinitely many prime numbers.
Intuition: No matter how many primes you list, you can always find another. The primes never "run out."
Proof (Euclid, ~300 BCE)
Suppose for contradiction there are only finitely many primes: \(p_1, p_2, \ldots, p_n\).
Construct \(Q = p_1 \cdot p_2 \cdots p_n + 1\).
For each \(p_i\): When we divide \(Q\) by \(p_i\), the remainder is 1 (since \(p_i \mid p_1 p_2 \cdots p_n\)). So \(p_i \nmid Q\).
Since \(Q > 1\), by the Fundamental Theorem of Arithmetic, \(Q\) has at least one prime factor.
But this prime factor cannot be any of \(p_1, \ldots, p_n\) (we just showed none of them divide \(Q\)).
This contradicts our assumption that \(p_1, \ldots, p_n\) were ALL the primes. \(\square\)
Triangle Inequality
Statement: For any \(x, y \in \mathbb{R}\): \(|x + y| \leq |x| + |y|\)
Intuition: The "direct path" is never longer than going via a detour. In a triangle, any side is at most the sum of the other two sides.
Proof

We know \(-|x| \leq x \leq |x|\) and \(-|y| \leq y \leq |y|\).

Adding: \(-(|x| + |y|) \leq x + y \leq |x| + |y|\).

This means \(|x + y| \leq |x| + |y|\). \(\square\)

Isomorphism for Finite-Dimensional Vector Spaces
Statement: Two finite-dimensional vector spaces \(V\) and \(W\) over the same field \(F\) are isomorphic if and only if \(\dim(V) = \dim(W)\).
Intuition: For finite-dimensional spaces, dimension completely determines the structure. \(\mathbb{R}^3\) is "the same as" any other 3-dimensional real vector space.
Proof

(\(\Rightarrow\)) Suppose \(T: V \to W\) is an isomorphism. Let \(\{v_1, \ldots, v_n\}\) be a basis for \(V\).

Then \(\{T(v_1), \ldots, T(v_n)\}\) is a basis for \(W\) (isomorphisms preserve linear independence and spanning). So \(\dim(V) = \dim(W) = n\).

(\(\Leftarrow\)) Suppose \(\dim(V) = \dim(W) = n\).

Let \(\{v_1, \ldots, v_n\}\) be a basis for \(V\) and \(\{w_1, \ldots, w_n\}\) be a basis for \(W\).

Define \(T: V \to W\) by \(T(v_i) = w_i\) and extend linearly.

\(T\) is bijective (maps basis to basis) and linear, so it's an isomorphism. \(\square\)

Proof Strategies

Knowing which technique to use is half the battle. Here are the main strategies with templates and when to use them.

Direct Proof

Goal: Prove \(P \Rightarrow Q\)

Template:
  1. Assume \(P\) is true.
  2. Use definitions, axioms, and known theorems...
  3. Conclude that \(Q\) is true.
When to use: When there's a clear logical path from hypothesis to conclusion. Often the first approach to try.
Example: Prove "If \(n\) is even, then \(n^2\) is even."

Assume \(n\) is even, so \(n = 2k\) for some integer \(k\).

Then \(n^2 = (2k)^2 = 4k^2 = 2(2k^2)\), which is even. \(\square\)

Proof by Contradiction

Goal: Prove statement \(P\)

Template:
  1. Assume \(\neg P\) (the negation of what you want to prove).
  2. Derive a logical contradiction.
  3. Conclude \(P\) must be true.
When to use: When proving something doesn't exist, when the statement involves "infinite," or when direct proof seems blocked. Look for phrases like "there is no..." or "infinitely many."
Example: Prove \(\sqrt{2}\) is irrational.

Assume \(\sqrt{2} = a/b\) in lowest terms. Then \(2b^2 = a^2\), so \(a^2\) is even, so \(a\) is even.

Write \(a = 2k\). Then \(2b^2 = 4k^2\), so \(b^2 = 2k^2\), so \(b\) is even.

Both \(a\) and \(b\) are even—contradicts "lowest terms." \(\square\)

Proof by Contrapositive

Goal: Prove \(P \Rightarrow Q\)

Template:
  1. Instead, prove \(\neg Q \Rightarrow \neg P\) (logically equivalent!)
  2. Assume \(\neg Q\).
  3. Derive \(\neg P\).
When to use: When the conclusion \(Q\) is hard to work with directly, but \(\neg Q\) gives useful information. Often used when \(Q\) involves existence and \(\neg Q\) gives you something concrete.
Example: Prove "If \(n^2\) is even, then \(n\) is even."

Contrapositive: "If \(n\) is odd, then \(n^2\) is odd."

Assume \(n\) is odd: \(n = 2k+1\). Then \(n^2 = 4k^2 + 4k + 1 = 2(2k^2 + 2k) + 1\), which is odd. \(\square\)

Mathematical Induction

Goal: Prove \(P(n)\) for all \(n \in \mathbb{N}\)

Template:
  1. Base Case: Prove \(P(1)\) (or \(P(0)\), depending on domain).
  2. Inductive Hypothesis: Assume \(P(k)\) is true for some arbitrary \(k\).
  3. Inductive Step: Prove \(P(k+1)\) using the hypothesis.
  4. Conclusion: By induction, \(P(n)\) holds for all \(n\).
When to use: Proving statements about all natural numbers, recursive definitions, summation formulas, divisibility results.
Strong Induction

Goal: Prove \(P(n)\) for all \(n \in \mathbb{N}\)

Template:
  1. Base Case(s): Prove \(P(1)\) (and possibly \(P(2), \ldots\)).
  2. Strong Hypothesis: Assume \(P(1), P(2), \ldots, P(k)\) are ALL true.
  3. Inductive Step: Prove \(P(k+1)\) using any/all of the previous cases.
When to use: When \(P(k+1)\) depends on cases before \(P(k)\), such as \(P(k-1)\) or \(P(k/2)\). Common in number theory and recursive algorithms.
Example: Every integer \(n \geq 2\) can be written as a product of primes.

Strong induction: Assume all integers from 2 to \(k\) have prime factorizations.

For \(k+1\): If prime, done. If composite, \(k+1 = ab\) where \(2 \leq a, b \leq k\).

By hypothesis, \(a\) and \(b\) have prime factorizations. Multiply them together. \(\square\)

Proof by Cases

Goal: Prove a statement by dividing into exhaustive cases

Template:
  1. Identify cases that cover all possibilities.
  2. Prove the statement in each case separately.
  3. Since cases are exhaustive, the statement holds universally.
When to use: When the situation naturally splits (even/odd, positive/negative/zero, etc.), or when different cases require different arguments.
Example: Prove \(n^2 + n\) is even for all integers \(n\).

Case 1: \(n\) is even. Then \(n^2\) is even and \(n\) is even, so sum is even.

Case 2: \(n\) is odd. Then \(n^2\) is odd and \(n\) is odd, so sum is even. \(\square\)

Proof by Definition

Goal: Prove something satisfies a definition

Template:
  1. State the precise definition of the concept.
  2. Verify each condition of the definition.
  3. Conclude the object satisfies the definition.
When to use: Proving something is an equivalence relation, a partition, continuous, a group, etc. Always start by unpacking the definition!
Example: Prove \(\sim\) defined by \(a \sim b \iff a - b \in \mathbb{Z}\) is an equivalence relation on \(\mathbb{R}\).

Reflexive: \(a - a = 0 \in \mathbb{Z}\) ✓

Symmetric: If \(a - b \in \mathbb{Z}\), then \(b - a = -(a-b) \in \mathbb{Z}\) ✓

Transitive: If \(a - b \in \mathbb{Z}\) and \(b - c \in \mathbb{Z}\), then \(a - c = (a-b) + (b-c) \in \mathbb{Z}\) ✓ \(\square\)
