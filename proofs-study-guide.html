Theorems (Statements & Proofs)

Cantor's Theorem

Statement: For any set \(A\), \(|A| < |\mathcal{P}(A)|\). (The cardinality of the power set is strictly greater).

Proof (Diagonalization)

Assume for contradiction there exists a surjection \(f: A \to \mathcal{P}(A)\).

Consider the set \(D = \{ x \in A \mid x \notin f(x) \}\). Since \(D \subseteq A\), \(D \in \mathcal{P}(A)\).

Because \(f\) is surjective, there must exist some \(d \in A\) such that \(f(d) = D\).

Ask: Is \(d \in D\)?

  • If \(d \in D\), then by definition of \(D\), \(d \notin f(d)\). Since \(f(d) = D\), this means \(d \notin D\). Contradiction.
  • If \(d \notin D\), then by definition of \(D\), \(d \in f(d)\). Since \(f(d) = D\), this means \(d \in D\). Contradiction.

Thus, no such function \(f\) can exist.

Fermat's Little Theorem

Statement: If \(p\) is prime and \(a\) is an integer, then \(a^p \equiv a \pmod p\).

Proof (Induction on \(a \ge 0\))

Base Case: If \(a=0\), \(0^p = 0 \equiv 0 \pmod p\). True.

Inductive Step: Assume \(k^p \equiv k \pmod p\). We show \((k+1)^p \equiv k+1 \pmod p\).

By the Binomial Theorem: \((k+1)^p = k^p + \binom{p}{1}k^{p-1} + \dots + \binom{p}{p-1}k + 1\).

Since \(p\) is prime, \(p\) divides \(\binom{p}{i}\) for all \(1 \le i \le p-1\). Thus all middle terms are \(\equiv 0 \pmod p\).

\((k+1)^p \equiv k^p + 1 \pmod p\). By hypothesis \(k^p \equiv k\), so \((k+1)^p \equiv k + 1 \pmod p\).

Russell's Paradox

Statement: There is no set of all sets.

Proof

Suppose \(U\) is the set of all sets. Define \(R = \{ A \in U \mid A \notin A \}\).

Ask: Is \(R \in R\)?

  • If \(R \in R\), then by definition of \(R\), \(R \notin R\). Contradiction.
  • If \(R \notin R\), then by definition of \(R\), \(R \in R\). Contradiction.

Thus, the definition of such a set \(U\) leads to a logical inconsistency.

Well-Ordering Principle

Statement: Every non-empty subset of \(\mathbb{N}\) has a least element.

Proof (Using Strong Induction)

Let \(S \subseteq \mathbb{N}\). We prove the contrapositive: If \(S\) has no least element, then \(S\) is empty.

Let \(P(n)\) be the statement "\(1, 2, \dots, n \notin S\)".

Base Case: \(1 \notin S\) (otherwise 1 would be the least element). Thus \(P(1)\) is true.

Inductive Step: Assume \(P(k)\) is true (i.e., \(1, \dots, k \notin S\)). Consider \(k+1\). If \(k+1 \in S\), since no smaller integer is in \(S\), \(k+1\) would be the least element. Since \(S\) has no least element, \(k+1 \notin S\). Thus \(P(k+1)\) is true.

By induction, \(\forall n \in \mathbb{N}, n \notin S\). Therefore \(S = \emptyset\).

Infinitude of Primes

Statement: There are infinitely many prime numbers.

Proof (Euclid)

Assume there are finitely many primes: \(p_1, p_2, \dots, p_n\).

Consider the integer \(Q = p_1 p_2 \dots p_n + 1\).

For any \(p_i\), \(Q \equiv 1 \pmod{p_i}\), so no \(p_i\) divides \(Q\).

Since \(Q > 1\), by the Fundamental Theorem of Arithmetic, \(Q\) must have a prime factor. But this factor cannot be any of our list \(p_1, \dots, p_n\). This contradicts the assumption that the list contained all primes.

Isomorphism for Finite Dim. Vector Spaces

Statement: Two finite-dimensional vector spaces \(V\) and \(W\) over the same field \(F\) are isomorphic if and only if \(\dim(V) = \dim(W)\).

Proof Sketch

\((\implies)\) If \(V \cong W\), an isomorphism maps a basis of \(V\) to a basis of \(W\), preserving the size (dimension).

\((\impliedby)\) Let \(\dim(V) = \dim(W) = n\). Let \(B_V = \{v_1, \dots, v_n\}\) be a basis for \(V\) and \(B_W = \{w_1, \dots, w_n\}\) be a basis for \(W\).

Define a linear transformation \(T: V \to W\) by \(T(v_i) = w_i\) and extending linearly. Since \(T\) maps a basis to a basis bijectively, \(T\) is an isomorphism.
