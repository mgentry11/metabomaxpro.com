Set Theory Symbols

Sets are collections of objects. These symbols describe membership, relationships between sets, and operations on sets.
Symbol Name Meaning Example
\(\in\) Element of \(x \in A\) means "\(x\) is an element of set \(A\)" \(3 \in \{1,2,3\}\)
\(\notin\) Not element of \(x \notin A\) means "\(x\) is not in set \(A\)" \(4 \notin \{1,2,3\}\)
\(\subseteq\) Subset \(A \subseteq B\) means "every element of \(A\) is in \(B\)" \(\{1,2\} \subseteq \{1,2,3\}\)
\(\subset\) Proper subset \(A \subset B\) means "\(A \subseteq B\) and \(A \neq B\)" \(\{1,2\} \subset \{1,2,3\}\)
\(\supseteq\) Superset \(A \supseteq B\) means "\(B\) is a subset of \(A\)" \(\{1,2,3\} \supseteq \{1,2\}\)
\(\cup\) Union \(A \cup B\) = elements in \(A\) OR \(B\) (or both) \(\{1,2\} \cup \{2,3\} = \{1,2,3\}\)
\(\cap\) Intersection \(A \cap B\) = elements in BOTH \(A\) AND \(B\) \(\{1,2\} \cap \{2,3\} = \{2\}\)
\(\setminus\) or \(-\) Set difference \(A \setminus B\) = elements in \(A\) but NOT in \(B\) \(\{1,2,3\} \setminus \{2\} = \{1,3\}\)
\(\emptyset\) or \(\{\}\) Empty set The set with no elements \(\{x : x \neq x\} = \emptyset\)
\(\mathcal{P}(A)\) Power set Set of all subsets of \(A\) \(\mathcal{P}(\{1,2\}) = \{\emptyset, \{1\}, \{2\}, \{1,2\}\}\)
\(|A|\) Cardinality Number of elements in \(A\) \(|\{a,b,c\}| = 3\)
\(A \times B\) Cartesian product Set of all ordered pairs \((a,b)\) \(\{1,2\} \times \{a,b\} = \{(1,a),(1,b),(2,a),(2,b)\}\)
\(\bigcup\) Big union Union of many sets \(\bigcup_{i=1}^{3} A_i = A_1 \cup A_2 \cup A_3\)
\(\bigcap\) Big intersection Intersection of many sets \(\bigcap_{i=1}^{3} A_i = A_1 \cap A_2 \cap A_3\)

Logic Symbols

These symbols express logical statements, quantifiers, and connectives essential for writing precise mathematical statements.
Symbol Name Meaning Example
\(\forall\) For all (Universal) "For every" or "for all" \(\forall x \in \mathbb{R}, x^2 \geq 0\)
\(\exists\) There exists (Existential) "There exists at least one" \(\exists x \in \mathbb{R}: x^2 = 2\)
\(\exists!\) Unique existence "There exists exactly one" \(\exists! x \in \mathbb{R}: x + 3 = 5\)
\(\nexists\) Does not exist "There does not exist" \(\nexists x \in \mathbb{R}: x^2 < 0\)
\(\Rightarrow\) Implies \(P \Rightarrow Q\): "if \(P\), then \(Q\)" \(x > 2 \Rightarrow x > 0\)
\(\Leftarrow\) Is implied by \(P \Leftarrow Q\): "\(P\) if \(Q\)" \(x > 0 \Leftarrow x > 2\)
\(\Leftrightarrow\) or \(\iff\) If and only if (iff) \(P\) is true exactly when \(Q\) is true \(x^2 = 4 \iff x = \pm 2\)
\(\neg\) or \(\sim\) Not (Negation) Opposite of the statement \(\neg(x > 0)\) means \(x \leq 0\)
\(\land\) And (Conjunction) Both statements are true \(x > 0 \land x < 5\)
\(\lor\) Or (Disjunction) At least one statement is true \(x < 0 \lor x > 0\)
\(\therefore\) Therefore Conclusion follows \(x = 2, \therefore x^2 = 4\)
\(\because\) Because Reason for conclusion \(x^2 = 4 \because x = 2\)
\(:\) or \(\mid\) or s.t. Such that Condition that follows \(\{x \in \mathbb{R} : x > 0\}\)

Relations & Equivalence

These symbols describe relationships between elements, including equivalence relations, orders, and congruence.
Symbol Name Meaning Example
\(=\) Equals Identical in value \(2 + 2 = 4\)
\(\neq\) Not equal Different values \(3 \neq 5\)
\(\sim\) Equivalent to Related by equivalence relation \(a \sim b\) means \(a\) equivalent to \(b\)
\(\equiv\) Congruent / Identical Congruent modulo \(n\), or identically equal \(7 \equiv 1 \pmod{3}\)
\([a]\) Equivalence class Set of all elements equivalent to \(a\) \([0] = \{\ldots,-3,0,3,6,\ldots\}\) in \(\mathbb{Z}_3\)
\(S/\!\sim\) Quotient set Set of all equivalence classes \(\mathbb{Z}/\!\equiv_3 = \{[0],[1],[2]\}\)
\(\leq\) Less than or equal Partial or total order \(3 \leq 5\)
\(\geq\) Greater than or equal At least as large \(5 \geq 3\)
\(<\) Less than Strictly smaller \(2 < 4\)
\(>\) Greater than Strictly larger \(4 > 2\)
\(\cong\) Isomorphic Same structure (bijection preserving operations) \((\mathbb{Z}_2, +) \cong (\{0,1\}, \oplus)\)
\(\approx\) Approximately equal Close in value \(\pi \approx 3.14159\)
\(\propto\) Proportional to Constant ratio \(y \propto x\) means \(y = kx\)
\(a \mid b\) Divides \(a\) divides \(b\) evenly (no remainder) \(3 \mid 12\) (12 = 3×4)
\(a \nmid b\) Does not divide \(a\) does not divide \(b\) evenly \(3 \nmid 10\)

Function Symbols

Functions map elements from one set to another. These symbols describe function properties and notation.
Symbol Name Meaning Example
\(f: A \to B\) Function mapping \(f\) is a function from set \(A\) to set \(B\) \(f: \mathbb{R} \to \mathbb{R}\)
\(f(x)\) Function value Output of \(f\) when input is \(x\) \(f(2) = 4\) if \(f(x) = x^2\)
\(x \mapsto f(x)\) Maps to \(x\) is sent to \(f(x)\) \(x \mapsto x^2\)
\(f^{-1}\) Inverse function Function that "undoes" \(f\) If \(f(x) = 2x\), then \(f^{-1}(x) = x/2\)
\(f \circ g\) Composition \((f \circ g)(x) = f(g(x))\) \((f \circ g)(x)\) means apply \(g\) first, then \(f\)
\(\text{dom}(f)\) Domain Set of valid inputs \(\text{dom}(\sqrt{x}) = [0, \infty)\)
\(\text{range}(f)\) or \(\text{im}(f)\) Range / Image Set of actual outputs \(\text{range}(x^2) = [0, \infty)\)
\(f^{-1}(B)\) Preimage Set of inputs that map into \(B\) \(f^{-1}(\{4\}) = \{-2, 2\}\) for \(f(x)=x^2\)
1-1 or injective One-to-one \(f(a) = f(b) \Rightarrow a = b\) \(f(x) = 2x\) is injective
onto or surjective Onto Every element in codomain is hit \(f: \mathbb{R} \to \mathbb{R}\), \(f(x) = x^3\) is onto
bijective Bijection Both injective and surjective \(f(x) = x^3\) on \(\mathbb{R}\) is bijective

Number Sets

Standard notation for common sets of numbers.
Symbol Name Meaning Elements
\(\mathbb{N}\) Natural numbers Counting numbers (positive integers) \(\{1, 2, 3, 4, \ldots\}\) or \(\{0,1,2,\ldots\}\)
\(\mathbb{Z}\) Integers Whole numbers (positive, negative, zero) \(\{\ldots, -2, -1, 0, 1, 2, \ldots\}\)
\(\mathbb{Z}^+\) Positive integers Integers greater than zero \(\{1, 2, 3, \ldots\}\)
\(\mathbb{Z}_n\) Integers mod n Equivalence classes of integers modulo \(n\) \(\mathbb{Z}_3 = \{0, 1, 2\}\)
\(\mathbb{Q}\) Rational numbers Fractions \(p/q\) where \(p, q \in \mathbb{Z}\), \(q \neq 0\) \(\frac{1}{2}, -\frac{3}{4}, 5\)
\(\mathbb{R}\) Real numbers All points on the number line \(\pi, \sqrt{2}, -3.5, 0\)
\(\mathbb{R}^+\) Positive reals Real numbers greater than zero \((0, \infty)\)
\(\mathbb{C}\) Complex numbers Numbers \(a + bi\) where \(i^2 = -1\) \(3 + 2i, -i, 5\)
\(\mathbb{R}^n\) n-dimensional space \(n\)-tuples of real numbers \(\mathbb{R}^2\) = plane, \(\mathbb{R}^3\) = 3D space

Operations & Other Symbols

Common mathematical operations, summation notation, and other useful symbols.
Symbol Name Meaning Example
\(\sum\) Summation Sum over a range \(\sum_{i=1}^{n} i = 1 + 2 + \cdots + n\)
\(\prod\) Product Product over a range \(\prod_{i=1}^{n} i = n!\)
\(n!\) Factorial \(n \cdot (n-1) \cdot \ldots \cdot 2 \cdot 1\) \(5! = 120\)
\(\binom{n}{k}\) Binomial coefficient "n choose k" = \(\frac{n!}{k!(n-k)!}\) \(\binom{5}{2} = 10\)
\(|x|\) Absolute value Distance from zero \(|-3| = 3\)
\(\lfloor x \rfloor\) Floor Greatest integer \(\leq x\) \(\lfloor 3.7 \rfloor = 3\)
\(\lceil x \rceil\) Ceiling Least integer \(\geq x\) \(\lceil 3.2 \rceil = 4\)
\(\gcd(a,b)\) GCD Greatest common divisor \(\gcd(12, 18) = 6\)
\(\text{lcm}(a,b)\) LCM Least common multiple \(\text{lcm}(4, 6) = 12\)
\(\infty\) Infinity Unbounded quantity \(\lim_{x \to \infty} 1/x = 0\)
\(\lim\) Limit Value approached \(\lim_{x \to 2} x^2 = 4\)
\(\min, \max\) Minimum, Maximum Smallest/largest value \(\min(3, 5) = 3\)

Proof Notation

Special notation commonly used in mathematical proofs.
Symbol Name Meaning Usage
\(\square\) or QED End of proof Proof is complete Write at end of proof
\(\Rightarrow\!\Leftarrow\) or ⊥ Contradiction Logical contradiction reached Used in proof by contradiction
WTS Want to show Statement we need to prove WTS: \(n^2\) is even
WLOG Without loss of generality Assume special case that covers all cases WLOG, assume \(a \leq b\)
IH Inductive hypothesis Assumption in induction proof By IH, \(P(k)\) is true
\(:=\) or \(\triangleq\) Defined as Left side is defined to equal right \(f(x) := x^2 + 1\)
iff If and only if Same as \(\Leftrightarrow\) \(A = B\) iff \(A \subseteq B\) and \(B \subseteq A\)
s.t. Such that Condition follows \(\exists x\) s.t. \(x^2 = 2\)
w.r.t. With respect to In relation to Continuous w.r.t. \(x\)

Greek Letters

Greek letters are commonly used for variables, constants, and special values in mathematics.
Symbol Name Common Uses Example
\(\alpha, \beta, \gamma\) alpha, beta, gamma Angles, coefficients, parameters Triangle angles
\(\delta, \Delta\) delta Small change, difference \(\delta\) in ε-δ proofs
\(\varepsilon, \epsilon\) epsilon Small positive number \(\varepsilon > 0\) in limits
\(\theta, \Theta\) theta Angles in trigonometry \(\sin\theta\)
\(\lambda\) lambda Eigenvalue, parameter \(Av = \lambda v\)
\(\mu\) mu Mean, measure Population mean \(\mu\)
\(\pi\) pi Ratio of circumference to diameter \(\pi \approx 3.14159\)
\(\sigma, \Sigma\) sigma Sum (Σ), std deviation (σ) \(\Sigma_{i=1}^n x_i\)
\(\phi, \varphi\) phi Golden ratio, function Euler's \(\varphi\) function
\(\omega, \Omega\) omega Angular frequency, sample space Big-\(\Omega\) notation
