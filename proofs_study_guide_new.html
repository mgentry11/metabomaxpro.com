1. Logic & Quantifiers

Understanding the language of mathematics is the first step.

Basic Logical Connectives
Implication (\(P \implies Q\))
False only when \(P\) is True and \(Q\) is False. Equivalent to \(\neg P \lor Q\).
Biconditional (\(P \iff Q\))
True when \(P\) and \(Q\) have the same truth value. Means \((P \implies Q) \land (Q \implies P)\).

Quantifiers
Universal Quantifier (\(\forall x \in S, P(x)\))
"For all \(x\) in \(S\), \(P(x)\) is true."
Disproved by finding a single counterexample.
Existential Quantifier (\(\exists x \in S, P(x)\))
"There exists at least one \(x\) in \(S\) such that \(P(x)\) is true."
Proved by finding a single example.

De Morgan's Laws for Logic

Negating quantifiers flips them:

  • \(\neg (\forall x, P(x)) \iff \exists x, \neg P(x)\)
  • \(\neg (\exists x, P(x)) \iff \forall x, \neg P(x)\)

2. Definitions

Master these definitions—they are the foundation of every proof.

Set Equality
Definition: \(A = B \iff A \subseteq B \text{ and } B \subseteq A\)

What This Really Means

Two sets are equal if and only if they contain exactly the same elements—no more, no less. The order doesn't matter, and duplicates don't count.

Key insight: To prove \(A = B\), you must prove TWO things:

  1. (\(A \subseteq B\)): Take any element \(x \in A\) and show \(x \in B\)
  2. (\(B \subseteq A\)): Take any element \(y \in B\) and show \(y \in A\)
    When A = B:                    When A ≠ B:

    ┌─────────────┐               ┌─────────┐
    │             │               │    A    │
    │   A = B     │               │  ┌──────┼──┐
    │  {1,2,3}    │               │  │  ∩   │  │ B
    │             │               └──┼──────┘  │
    └─────────────┘                  └─────────┘

    Same elements                  Different elements
Sets are equal when they overlap completely
Example 1: Prove \((\{x \in \mathbb{Z} : x^2 = 1\}) = \(\{-1, 1\}\)

Proof:

(\( \subseteq \)): Let \(x\) be in the left set. Then \(x \in \mathbb{Z}\) and \(x^2 = 1\).
Solving: \(x^2 - 1 = 0 \Rightarrow (x-1)(x+1) = 0 \Rightarrow x = 1\) or \(x = -1\).
So \(x \in \{-1, 1\}\). ✓

(\( \supseteq \)): Check: \(1^2 = 1\) ✓ and \((-1)^2 = 1\) ✓.
So both elements of \(\{-1, 1\}\) satisfy the condition. ✓

Partition
Definition: A partition of set \(S\) is a collection \(\{A_i\}_{i \in I}\) such that:
  1. \(A_i \neq \emptyset\) for all \(i\) (each part is non-empty)
  2. \(A_i \cap A_j = \emptyset\) for \(i \neq j\) (parts are pairwise disjoint)
  3. \(\bigcup_{i \in I} A_i = S\) (parts cover all of \(S\))

What This Really Means

A partition "cuts up" a set into non-overlapping pieces that together form the whole set. Think of:

  • Pizza slices: Each slice is non-empty, slices don't overlap, all slices together = whole pizza

Equivalence Relation
Definition: A relation \(\sim\) on set \(S\) is an equivalence relation if:
  1. Reflexive: \(a \sim a\) for all \(a \in S\)
  2. Symmetric: \(a \sim b \Rightarrow b \sim a\)
  3. Transitive: \(a \sim b\) and \(b \sim c \Rightarrow a \sim c\)

What This Really Means

An equivalence relation groups elements that are "the same" in some way. Think of it as a generalized notion of equality.

Partial Order
Definition: A relation \(\leq\) on set \(S\) is a partial order if:
  1. Reflexive: \(a \leq a\)
  2. Antisymmetric: \(a \leq b\) and \(b \leq a \Rightarrow a = b\)
  3. Transitive: \(a \leq b\) and \(b \leq c \Rightarrow a \leq c\)

3. Axioms & Principles

These fundamental truths are accepted without proof and form the basis for all mathematical reasoning.

Principle of Mathematical Induction
Let \(P(n)\) be a statement. If:
  1. Base: \(P(1)\) is true
  2. Step: \(P(k) \Rightarrow P(k+1)\) for all \(k \geq 1\)
Then \(P(n)\) is true for all \(n \in \mathbb{N}\). 
    Domino Effect:

    P(1) → P(2) → P(3) → P(4) → P(5) → ...
      ↓      ↓      ↓      ↓      ↓
    TRUE   TRUE   TRUE   TRUE   TRUE   ...
Well-Ordering Principle
Every non-empty subset of \(\mathbb{N}\) has a least element.
Axiom of Choice & Zorn's Lemma
Axiom of Choice: Given any collection of non-empty sets, there exists a choice function that selects one element from each set.
Zorn's Lemma: If every chain in a partially ordered set \(P\) has an upper bound in \(P\), then \(P\) contains at least one maximal element.

4. Key Theorems

Master these proofs—they demonstrate important techniques.

Fundamental Theorem of Arithmetic
Every integer \(n > 1\) can be represented uniquely as a product of prime numbers, up to the order of the factors.

Proof Strategy: Existence via Strong Induction; Uniqueness via Euclid's Lemma.

Strong Induction
If \(P(1)\) is true, and for any \(k \ge 1\), \((P(1) \land P(2) \land \cdots \land P(k)) \implies P(k+1)\) is true, then \(P(n)\) is true for all \(n \in \mathbb{N}\).
Intuition: Similar to weak induction, but allows you to assume the truth of *all* previous cases, not just the immediately preceding one. It is logically equivalent to the Principle of Mathematical Induction and the Well-Ordering Principle.
Example: Every integer \(n \geq 2\) has a prime factorization.

Base: 2 is prime, so its factorization is just "2". ✓

Step: Assume all integers from 2 to \(k\) have prime factorizations.

For \(k+1\): If prime, done. If composite, \(k+1 = ab\) where \(2 \leq a, b \leq k\).

By hypothesis, \(a\) and \(b\) have factorizations. Multiply them. ✓ ∎

Cantor's Theorem
For any set \(A\): \(|A| < |\u0002P(A)|\)
Proof (Diagonalization)

Assume surjective \(f: A \to \mathcal{P}(A)\). Define \(D = \{x \in A : x \notin f(x)\}).

Since \(f\) surjective, \(\exists d: f(d) = D\). Then \(d \in D \iff d \notin D\). Contradiction! ∎

Infinitude of Primes (Euclid)
There are infinitely many prime numbers.
Proof

Suppose finite list \(p_1, \dots, p_n\). Let \(Q = p_1 \dots p_n + 1\). Then \(Q\) is not divisible by any \(p_i\). Contradiction. ∎

Bézout's Identity
Let \(a, b \in \mathbb{Z}\) not both zero. Then \(\exists x, y \in \mathbb{Z}\) such that \(ax + by = \gcd(a, b)\).
Pigeonhole Principle
If \(n+1\) items are put into \(n\) containers, at least one container has \(\ge 2\) items.

Direct Proof
To prove \(P \Rightarrow Q\): Assume \(P\). Use definitions/logic. Conclude \(Q\).
Example: If \(n\) is even, then \(n^2\) is even.

Assume \(n=2k\). Then \(n^2 = 4k^2 = 2(2k^2)\). ∎

Proof by Contradiction
To prove \(P\): Assume \(\neg P\). Derive contradiction.
Example: \(\sqrt{2}\) is irrational.

Assume \(\sqrt{2}=a/b\) (lowest terms). Then \(2b^2=a^2 \implies a\) even \(\implies b\) even. Contradiction. ∎

Proof by Contrapositive
To prove \(P \Rightarrow Q\): Prove \(\neg Q \Rightarrow \neg P\).
Example: If \(n^2\) is even, then \(n\) is even.

Contrapositive: If \(n\) is odd, \(n^2\) is odd. (True). ∎

Proof by Cases
Divide domain into exhaustive cases. Prove true for each.
Example: \(|x| \ge 0\) for all \(x \in \mathbb{R}\).

Case 1: \(x \ge 0 \implies |x|=x \ge 0\).
Case 2: \(x < 0 \implies |x|=-x > 0\). ∎
